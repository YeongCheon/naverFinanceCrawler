## naverFinanceCrawler

네이버에서 제공하는 주가정보를 수집하는 수집기입니다.

### environment

* OS: ubuntu 16.04 LTS
* language: [go 1.8](https://golang.org)
* db: [elasticsearch6](https://www.elastic.co)
* editor: emacs24
 
 ### how to use
 
 ```bash
 go run main.go
 ```
 
 ### other
 `data.csv`는 한국거래소에서 제공하는 KOSPI 상장사 목록입니다.[링크](http://marketdata.krx.co.kr/mdi#document=040601)
